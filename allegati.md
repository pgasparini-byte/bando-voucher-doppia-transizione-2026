## Allegato B — Prospetto delle spese (Excel)

L'Allegato B e' un file Excel guidato. Ecco come e' strutturato:

### Sezioni del file

| Sezione | Cosa contiene |
|---------|---------------|
| **Dati impresa** | Nome impresa, codice fiscale, provincia della sede dell'intervento |
| **a) Consulenza** | 3 righe per i servizi di consulenza. Ogni riga ha un menu a tendina per selezionare la sottocategoria (analisi, piani sviluppo, implementazione tecnologie, sistemi gestione, matching ricerca, Innovation Manager, certificazioni ESG) |
| **b) Formazione** | 2 righe per i servizi di formazione |
| **c) Tecnologie** | 4 righe per l'acquisto di tecnologie. Ogni riga ha un menu a tendina con l'elenco completo delle tecnologie ammissibili (AI, cloud, IoT, ERP, CRM, ecc.) |
| **Calcolo automatico** | Il file verifica automaticamente: investimento minimo (4.000 EUR), range 30-70% per consulenza+formazione, contributo (50%) |

### Come lo gestiamo

> **Turbo Crowd compila l'Allegato B per te.** Inseriamo i nostri dati (ragione sociale, codice fiscale), selezioniamo le voci di spesa corrette e la ripartizione degli importi. Te lo inviamo gia' pronto: lo controlli, lo firmi e lo carichi.

## Allegato C — Calcolo dimensione d'impresa (Excel)

L'Allegato C serve a dimostrare che la tua impresa rientra nella definizione di Micro, Piccola o Media Impresa. E' un file Excel con formule automatiche.

### Cosa devi compilare

| Sezione | Cosa inserire |
|---------|---------------|
| **Sezione 1** | Ragione sociale, codice fiscale, anno di riferimento dell'ultimo bilancio approvato |
| **Dati economici** | Fatturato, numero di occupati e totale attivo degli **ultimi 3 bilanci approvati** (anno t, t-1, t-2) |
| **Tipo impresa** | Seleziona dal menu a tendina se sei impresa **autonoma**, **associata** o **collegata** |
| **Sezione 2** | Da compilare **solo** se sei un'impresa collegata o associata: inserisci i dati delle imprese del gruppo |

Il file calcola automaticamente la dimensione (MICRO, PICCOLA, MEDIA o GRANDE).

> **Solo le Micro, Piccole e Medie Imprese possono accedere al bando.** Se il risultato e' "GRANDE", la tua impresa non e' ammissibile.
