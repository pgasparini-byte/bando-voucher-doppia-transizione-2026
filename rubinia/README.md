| **Shopify** | E-commerce (integrazione CRM) | lett. k |
| **Klaviyo** | Marketing automation e newsletter | lett. k |
| **Postiz** | Social media management | lett. k |
| **eSolver + Extranet** | ERP e portale clienti (integrazione) | lett. k |

### Integrazioni principali

1. HubSpot → NocoDB → Twenty CRM (migrazione dati completa)
2. Shopify → Twenty CRM (allineamento clienti, ordini, garanzie)
3. eSolver/Extranet → Twenty CRM (storico riparazioni, garanzie)
4. Shopify → Klaviyo (segmentazione clienti per newsletter e referral)
5. Twenty CRM → Klaviyo (attivazione flussi su eventi CRM)
6. Twenty CRM → Postiz (sincronizzazione dati cliente per social)
7. Twenty CRM → Looker Studio (dashboard in tempo reale)
