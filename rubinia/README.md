4. Shopify → Klaviyo (segmentazione clienti per newsletter e referral)
5. Twenty CRM → Klaviyo (attivazione flussi su eventi CRM)
6. Twenty CRM → Postiz (sincronizzazione dati cliente per social)
7. Twenty CRM → Looker Studio (dashboard in tempo reale)
